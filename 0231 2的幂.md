#### [231. 2的幂](https://leetcode-cn.com/problems/power-of-two/)

难度简单260

给定一个整数，编写一个函数来判断它是否是 2 的幂次方。

**示例 1:**

```
输入: 1
输出: true
解释: 20 = 1
```

**示例 2:**

```
输入: 16
输出: true
解释: 24 = 16
```

**示例 3:**

```
输入: 218
输出: false
```



# 位运算

特别注意，这种位运算计算1的个数，只能应用在非负数上。

```c++
class Solution {
public:
    bool isPowerOfTwo(int n) {
        if (n < 0)
            return false;
        int count = 0;
        while (n & -n) {
            n &= (n - 1);
            ++count;
        }
        return count == 1;
    }
};
```
